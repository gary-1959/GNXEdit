Introduction

GNXEdit is an editor and librarian program for use with the DigiTech GNX1 guitar effects processor.

The functionality of the program is similar to the DigiTech GENEdit program for Windows, but includes enhanced library functionality and is cross-platform (Windows and Linux).

The program is written in Python and leverages the QT GUI framework. It iscompletely open source and freely available at {APP_GITHUB_LINK}.

System Exclusive

There was no information available from DigiTech detailing the System Exclusive commands, so to write the program the System Exclusive protocol had to be reverse engineered by snooping on communications between the GENEdit program and a GNX1 device. This work has been documented separately and is available in another GitHub repository: https://github.com/gary-1959/Digitech-GNX1-System-Exclusive.

There are gaps in this knowledge and any contribution to filling these gaps (or errors) will be most gratefully received and incorporated imto the document.

Installation

Windows

  1. Download and install Python (minimum version 13.2) if not already installed from https://www.python.org/downloads/windows
  2. Download and install the GitHub comman line interface (CLI) from https://github.com/cli/cli/releases/download/v2.65.0/gh_2.65.0_windows_386.zip
  3. Open a command window, create a local folder called "GNXEdit" and cd into it
  4. Create a sub-folder of GNXEdit and call it "program". Clone the GitHub repository at {APP_GITHUB_LINK} into it with the following command:gh repo clone gary-1959/GNXEdit
  5. Python works best when applications are run in their own dedicated environment. A Python environment is simply a folder which contains the main Python program and all the dependant libraries. Create a folder for your Python environment with the following command: python -m venv env Select the version 13.2 (or later) Python interpreter if prompted.
  6. Activate the Python environment with the command env\Scripts\Activate.ps1
  7. Install the dependant libraries with the command pip install -r requirements.txt
  8. Deactivate the Python environment with the command deactivate
